# Social Media App

A full-stack social media application built with **Next.js**, **MongoDB**, **NextAuth**, and **ImageKit**.  
It supports user authentication, media uploads, and a modern UI for sharing posts.

---

## 🚀 Tech Stack
- **Frontend:** Next.js 16, React, Tailwind CSS
- **Backend:** Node.js, Express (API routes in Next.js)
- **Database:** MongoDB Atlas
- **Auth:** NextAuth.js (OAuth + Credentials)
- **Media Storage:** ImageKit.io

---
⚠️ **Status:** This project is currently a work in progress. Features and configurations are actively being developed.
